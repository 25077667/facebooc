# Facebooc

Proof-of-concept Facebook clone in C.
The only dependency is SQLite3.

# Prerequisites
Install docker 

# Build and Run
```bash
cd facebooc
docker network create swag_facebooc
docker compose up
```
Then go to `localhost:8080` to view the deployment
