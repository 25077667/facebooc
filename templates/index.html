What is facebooc

Facebooc is a social platform designed by c language. Because of the basic of c, the platform can respond more faster than other platform. This verifies the application power of C language.

The website is designed using boostrap model. It is using scss instead of css language about the stylesheet. Because using scss, the stylesheet can be edit and recognize much more easier.

The database we use is SQLite3. SQLite is support to mostly device like Linux, Max and Windows. It is also easy to use while it provides an easy use API.

About Boostrap About SQLite3

Prerequisites

  1. Install SQLite development package:
    sudo apt-get install libsqlite3-dev
  2. Build and Run:
    cd facebooc
    make run
